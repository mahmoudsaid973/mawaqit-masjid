# Contributing
This guide will help you contribute to the Mawaqit Masjid project.
## Prerequisites
* Node.js 16 or higher
* PostgreSQL 13 or higher
* npm 8 or higher
## Contribution Steps
1. Fork the repository: `git fork https://github.com/username/mawaqit-masjid.git`
2. Create a new branch: `git checkout -b feature/your-feature`
3. Make changes and commit: `git commit -m "Your commit message"`
4. Push changes to your fork: `git push origin feature/your-feature`
5. Create a pull request: `git pull-request https://github.com/username/mawaqit-masjid.git`
## Code of Conduct
* Be respectful and considerate of others
* Follow the project's coding standards and best practices
* Test your changes thoroughly before submitting a pull request
