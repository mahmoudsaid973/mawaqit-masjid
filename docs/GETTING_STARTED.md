# Getting Started
This guide will help you get started with the Mawaqit Masjid project.
## Prerequisites
* Node.js 16 or higher
* PostgreSQL 13 or higher
* npm 8 or higher
## Installation
1. Clone the repository: `git clone https://github.com/username/mawaqit-masjid.git`
2. Install dependencies: `npm install`
3. Create a PostgreSQL database and update the `database` section in `src/db/schema.ts`
4. Run the application: `npm run dev`
5. Open the application in your web browser: `http://localhost:3000`
## Troubleshooting
* If you encounter any issues during installation, please refer to the [Troubleshooting Guide](docs/TROUBLESHOOTING.md).
