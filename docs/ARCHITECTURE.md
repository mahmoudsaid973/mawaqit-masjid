# Architecture
This document provides an overview of the system architecture.
## System Components
* Frontend: Next.js 15
* Backend: Node.js + Express
* Database: PostgreSQL
* Authentication: NextAuth
## System Flow
1. User requests the application
2. Next.js handles the request and renders the page
3. Express handles API requests and interacts with the database
4. PostgreSQL stores and retrieves data
5. NextAuth handles authentication and authorization
