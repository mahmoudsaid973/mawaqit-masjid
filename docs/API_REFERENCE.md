# API Reference
This document provides an overview of the API endpoints and their usage.
## Endpoints
### Prayer Times
* `GET /prayer-times`: Retrieves the current prayer times
* `POST /prayer-times`: Creates a new prayer time entry
### Events and Khutbah Topics
* `GET /events-and-khutbah-topics`: Retrieves a list of upcoming events and khutbah topics
* `POST /events-and-khutbah-topics`: Creates a new event or khutbah topic entry
### Quran Reader
* `GET /quran-reader`: Retrieves the Quran text for a given surah and ayah
* `POST /quran-reader`: Creates a new Quran reader entry
### Hadith Browser
* `GET /hadith-browser`: Retrieves a list of Hadiths for a given book and chapter
* `POST /hadith-browser`: Creates a new Hadith entry
### Azkar
* `GET /azkar`: Retrieves a list of Azkar for a given category
* `POST /azkar`: Creates a new Azkar entry
### Admin Portal
* `GET /admin-portal`: Retrieves the admin portal dashboard
* `POST /admin-portal`: Creates a new admin portal entry
