# Mawaqit Masjid
A comprehensive platform for managing prayer times, events, and khutbah topics, as well as providing a Quran reader, Hadith browser, and Azkar.
## Features
* Prayer Times
* Events and Khutbah Topics
* Quran Reader
* Hadith Browser
* Azkar
* Home Screen Widgets
* Admin Portal
## Getting Started
1. Clone the repository: `git clone https://github.com/username/mawaqit-masjid.git`
2. Install dependencies: `npm install`
3. Create a PostgreSQL database and update the `database` section in `src/db/schema.ts`
4. Run the application: `npm run dev`
5. Open the application in your web browser: `http://localhost:3000`
